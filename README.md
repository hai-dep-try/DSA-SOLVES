# DSA Solutions

A structured repository for all my Competitive Programming and Data Structures & Algorithms (DSA) solutions.

## Structure

- `00-Templates/`: Standard templates for C++ and Python.
- `01-Math-Introductory/`: Basic Math and Introductory problems (e.g. from CSES).
- `02-Array-TwoPointers/`: Array manipulation, Two Pointers technique.
- `03-Graph-BFS-DFS/`: Graph traversal algorithms (BFS, DFS).
- `04-Backtracking-Permutations/`: Backtracking and combinations/permutations.
- `05-Data-Structures/`: Custom data structures like Segment Tree, DSU, etc.
- `06-Strings/`: String algorithms and manipulation.
- `07-Disjoint-Set-Union/`: DSU algorithms and problems.

## Nền tảng Online Judge (OJ)
Các bài tập trong repository này được tổng hợp từ nhiều nền tảng lập trình thi đấu phổ biến:
- LeetCode
- Codeforces
- VNOJ
- CSES
- HackerRank
